    public async autoValidateScreens(): Promise since v0.106.* source

    Starts auto validation of screens. Screens are open one by one in sequence automatically until last visible screen. Auto validation starts from currently opened screen.

    Return:

    Promise

    A Promise of operation completion if implementation contains any asynchronous operations and/or call to superclass.autoValidateScreens. If implementation is synchronous you may return either resolved Promise or undefined.
